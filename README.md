## :sun_with_face: Atlas :cloud:

> A javascript powered weather Applcation.

#### Installation
```
git clone git@github.com:mosaaleb/Atlas.git
cd Atlas
npm install

# To build the app for production, run the following command:
npm run build-prod
```
